# 🎵 Spotify Song Downloader

A modern and responsive Spotify Song Downloader UI built using **HTML, CSS, and JavaScript**. This project validates Spotify song links, displays song details, and simulates the download process with an animated progress bar.

## ✨ Features

- 🎵 Spotify URL Validation
- 📱 Responsive Design
- 📊 Download Progress Animation
- 🔔 Success & Error Messages
- 📈 Download Statistics
- 🎨 Modern Spotify-inspired UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Use

1. Enter a valid Spotify track link.
2. Click **Download Song**.
3. View song details and simulated download progress.

> **Note:** This project is a frontend demo. To enable real downloads, integrate a supported API (such as a RapidAPI music service) with your own API key.
